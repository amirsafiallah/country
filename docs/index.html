Country

GitHub / Docker

Country is an IP-to-country geolocation API that returns a user’s country based on their IP address.

We offer a free instance—no API key needed. You can also self-host if preferred.

Usage

The API has a minimal interface.

GET /
Returns the country of the IP making the request, typically the user’s browser or app.
GET /{ip}
Returns the country of any given IP. The API supports both with IPv4 and IPv6.
GET /info
Provides metadata about the API, including when the data sources were last updated.

The API automatically updates MaxMind data every 24 hours in the background.

Notes

  • The API uses geolocation data provided by MaxMind and optionally Cloudfare.
  • Since 30 December 2019, you need to register for a license key to download the MaxMind data if you're self-hosting.
  • Our public instance rate-limits queries from the same IP at 10 requests per second. If you need to make more requests server-side, consider self-hosting or querying the MaxMind data directly.
  • The API does not log any requests.
